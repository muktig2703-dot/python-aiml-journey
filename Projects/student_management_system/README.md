# Student Management System
A simple menu-driven **Student Management System** built using Python. This project demonstrates the practical application of Object-Oriented Programming (OOP), lists, loops, conditional statements, and exception handling.
It allows users to add, view, and search student records through an interactive command-line interface.

## Features
Add new students
View all student records
Search students by name (case-insensitive)
Input validation using exception handling
Menu-driven interface
Built using Object-Oriented Programming (OOP)

## Technologies Used
Python 3
Object-Oriented Programming (OOP)
Classes & Objects
Lists
Loops
Conditional Statements
